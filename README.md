# EdgeViewer
Edge Embdedded webviewer.

Uses Microsoft's Edge WebView to have a very simple webviewer - https://www.nuget.org/packages/Microsoft.Web.WebView2

Call with a file or an URL as parameter.
Can also be invoked with a .url file, then it loads the associated icon as well. Otherwise it loads the local file's icon or displays the globe icon for websites.
